Medical Test Manager is a fullstack web application for organizing and accessing medical test records in a clinical environment.

It was built to simulate real-world backend architecture and microservice communication.
## Features

- Secure user management with Keycloak
- Microservice architecture using Spring Boot
- RESTful APIs secured with JWT
- Data persistence with PostgreSQL and MongoDB
- Dockerized environment with docker-compose
- Real-time communication with Kafka
- Doctor, Hospital and patient roles
- Modular and scalable codebase
